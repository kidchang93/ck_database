## Ⅴ) 프로젝트를 마치며 느낀 점

배우기 시작한지 얼마 안되서 함수를 사용하는 것이 자연스럽지 못했던 것 같아 아쉽다.
함수를 다 외우진 않고 찾아보는것에 좀 시간을 할당했다면
구조를 더 이해하는데 집중하고 시간을 많이 투자했다.
그로인해 함수가 눈에 들어오고 In 과 Out 이 명확해져 이해하기 좀더 수월했다.
함수 사용이 좀 더 원활해 지려면 코드를 보다 많이 작성하는 방법 밖에는 없을 것 같다.
